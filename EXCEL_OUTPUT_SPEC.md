# 엑셀 출력 명세 & 최종 검수 포인트

---

## 엑셀 리포터 출력 요구사항

### 순위
- **1등 ~ 30등**까지 정렬하여 출력

### 포함 컬럼
| 컬럼 | 설명 |
|------|------|
| 순위 | 1~30 |
| 제목/헤드라인 | 수집된 콘텐츠 제목 |
| 추천 점수 | 어그로 점수 (높은 순 정렬) |
| 출처 | 유튜브 / 구글뉴스 / 네이버뉴스 |
| **유튜브 URL** | 유튜브 영상인 경우 링크 |
| **뉴스 기사 URL** | 뉴스 기사인 경우 링크 |
| 조회수 | (유튜브) |
| 업로드/발행일 | |

> ⚠️ **필수**: 관련 유튜브 주소 또는 뉴스 기사 URL을 반드시 엑셀 칸에 기재

---

## 최종 검수 포인트 (수동 선정 기준)

엑셀 파일이 생성되면, 사용자가 아래 기준에 따라 **수동으로 최종 주제**를 선정합니다.

### 1. 시각화 가능성
> "하얀 구체 머리 캐릭터가 이 상황을 **도슨트처럼 설명하기 좋은 주제**인가?"

- 스토리텔링이 가능한가
- 시각 자료(차트, 그래프, 캐릭터)로 설명하기 적합한가

### 2. 확장성
> "이 주제로 **10분 이상** 이야기를 끌고 갈 **숫자 데이터**가 충분한가?"

- 추가 조사/통계 자료 확보 가능한가
- 논점을 펼칠 수 있는 구체적 수치가 있는가
